calendar-dialog

（一个日历弹框小组件）

引入

首先你需要通过命令行安装，就像这样： 

npm install --save calendar-dialog

安装好了以后，引入即可： 

import CalendarDialog from 'calendar-dialog';

当然，你也可以通过CDN的方式引入： 

<script src='https://cdn.jsdelivr.net/npm/calendar-dialog@0'></script>

语法 

CalendarDialog({
    year: number,
    month: number,
    day: number
}).then(function (time) {

    console.log(time);

});

例子

让我们举个例子：右边这个输入框就是运行的效果： （你可以点击一下试试效果）。

你直接调用上面的选择方法即可： 

CalendarDialog({
    year: 2021,
    month: 12,
    day: 1
}).then(function (time) {
    document.getElementById('input').value = time.year + "-" + time.month + "-" + time.day;
});
